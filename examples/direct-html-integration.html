Rock Academy Popup - Direct HTML Integration

Setup Instructions

To use this integration method:

  1. Add the configuration script before </body>
  2. Update the clientId with your GA4 client ID
  3. Update the CSS and JS URLs to point to your hosted files
  4. Test the popup appears on page load

Getting GA4 Client ID

To get the GA4 client_id dynamically, see the dynamic-loading.html example.

Or use GTM integration for automatic client_id retrieval.
